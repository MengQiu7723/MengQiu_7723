# 35前端

## 克隆项目到本地
```
git clone git@github.com:MengQiu7723/MengQiu_7723.git
```
## 安装项目依赖
```
npm install
```

## 编绎和热更新（用于开发环境）
```
npm run serve
```

### 编译并压缩（用于生产环境）
```
npm run build
```

### 检查并修复源文件
```
npm run lint
```

### 自定义配置
请查看 [Configuration Reference](https://cli.vuejs.org/config/).
